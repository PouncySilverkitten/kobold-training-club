Hello and welcome to Kobold Training Club!

You should find it fairly easy to use - filter the monsters on the table by using the different selectors on the left. You can search the results, and sort them by each column in the table.

Kobold Training Club also accepts sheets from Kobold Fight Club without needing any modification! Open the "Alignments" section to the left for more information.

Version notes:

  • Fixed (and added comprehensive testing for) a funky little bug about how monsters with non-unique items are handled. Expected behaviour now is that if a third-party monster shares a name with a first-party monster, the third-party monster will have the acronym of the source it's in added to its name. This is to avoid any confusion between monsters.
  • Sorted another annoying little bug which prevented CR30 monsters from being shown on occasion

The next few updates are likely to focus on back-end stuff - specifically to do with handling 3rd-party content better.

Thanks again, and happy rolling!
